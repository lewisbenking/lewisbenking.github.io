JimBot is an embodied conversational agent. This means he has a lip-syncing virtual human and UI frontend with a chatbot backend. He acts as a personal trainer that takes the user preferences and displays suitable timed workouts. These workouts include visual demonstrations and muscles affected diagrams. The chatbot was made using the DialogFlow framework by Google, and the character was created using Adobe Fuse and Mixamo. Unity was used to create the application, and was coded in C#.

MovieSearcher is a C# Windows Form application that connects to an open-source MongoDB data store to retrieve and display information about movies based on the user input. Users can search for specific titles, search by actor, search for a random title or view the top rated titles on the data store.

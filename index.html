JimBot

JimBot is an embodied conversational agent (a virtual human chatbot). He's a virtual personal trainer and asks for user preferences before providing suitable timed workouts, including visual demonstrations. The chatbot was made using DialogFlow by Google and the character was created using Adobe Fuse and Mixamo. Unity and C# was used to create the application.



MovieSearcher

MovieSearcher is a C# Windows Form application that connects to an open-source MongoDB data store to retrieve and display information about movies based on the user input. Users can search for specific titles, search by actor, search for a random title or view the top rated titles on the data store.

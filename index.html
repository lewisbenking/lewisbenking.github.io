Recent Projects



JimBot

JimBot was my submission for my Individual Project (dissertation) module at university. JimBot is a Unity app (C#) and is an embodied conversational agent illustrated with a virtual human character inside the application. JimBot asks for user preferences before displaying suitable timed workouts, including visual demonstrations of the exercises.



MovieSearcher

MovieSearcher was my coursework submission for the Advanced Databases module at university. We were instructed to create a (not fully-fledged) data-driven application for big data/open source data. MovieSearcher is a C# WinForms application that connects to a MongoDB data store to retrieve and display information about movies based on the user input. Users can search for specific titles, search by actor, search for a random title or view the top rated titles on the data store.
